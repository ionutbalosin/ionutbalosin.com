# Corporate Training Catalog

> Hands-on, instructor-led technical training for engineering organizations — designed to move teams from *knowing* to *applying*. Every course is workshop-heavy, runs on real code and real architectural decisions, and is updated each year to reflect what production engineering actually looks like (including the impact of AI on how we build and secure software).

---

## Java Performance Tuning

A deep, practical tour of the JVM — architecture, memory structure, garbage collectors, JIT compilation, tuning, profiling, and micro-benchmarking. Attendees leave with the mental model and the toolbox to *measure first, optimize second*, and to spot performance problems before they reach production.

- **Duration**: 16 hours
- **Details**: [Java Performance Tuning](https://ionutbalosin.com/training/java-performance-tuning/)

---

## Software Architecture Essentials — Designing for the AI Era

The fundamentals of software architecture — quality attributes, tactics, views, ADRs, technology evaluation — combined with what's changed in the AI era: agentic patterns, fitness functions, and AI as both a tool for the architect and a component running inside the systems we design. With all the new concerns that brings: non-determinism, cost, hallucinations, prompt injection.

- **Duration**: 16 hours
- **Details**: [Software Architecture Essentials — Designing for the AI Era](https://ionutbalosin.com/training/software-architecture-essentials/)

---

## Designing High-Performance, Scalable, and Resilient Applications — Including AI-Powered Systems

Architectural tactics, patterns, and styles for systems where performance, scalability, and resilience are first-class quality attributes — from sync/async communication and resilience patterns to microservices, event-driven, and reactive systems. Now extended with a dedicated track on designing systems that embed AI components: token cost as a runtime concern, LLM latency budgets, hallucinations as a design constraint, prompt injection as an attack surface.

- **Duration**: 16 hours
- **Details**: [Designing High-Performance, Scalable, and Resilient Applications — Including AI-Powered Systems](https://ionutbalosin.com/training/designing-high-performance-scalable-resilient-applications/)

---

## Application Security for Java Developers

Security as a first-class quality attribute, not an afterthought. Covers the classic landscape (OWASP Top 10, authentication and authorization with OAuth 2.0 / OpenID Connect, secure coding, security testing with SAST/DAST/SCA) and the new attack surface introduced by AI-augmented development and AI-powered applications (prompt injection, MCP/tool threat modelling, agent permissions, trust boundaries for LLM components in production).

- **Duration**: 14 hours
- **Details**: [Application Security for Java Developers](https://ionutbalosin.com/training/application-security-for-java-developers-course/)
